# ADR-005: Mobile-Web-First Product

## Status

Accepted.

## Decision

Pobal is a mobile-web-first hosted SaaS application.

Primary UX:

- smartphone browser;
- quick capture;
- today view;
- voice updates;
- meeting preparation;
- follow-up decisions.

Secondary UX:

- desktop browser for heavier workflows, admin, exports, billing, integration setup, and bulk review.

## Rationale

The core user is a consultant who needs to capture and use relationship intelligence between meetings, while travelling, and shortly before/after conversations.

## Consequences

- Navigation, screens, and flows must be designed mobile-first.
- Desktop must not drive the information architecture.
- Tables should not be the default mobile interaction pattern.
